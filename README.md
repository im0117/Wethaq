# Wathaq (وثاق) – Legal Services Application

## 📌 Overview
Wathaq is a full-stack web application developed to provide a structured platform for legal services. The system aims to help users access legal assistance through a simple, organized, and user-friendly interface.

This project was built entirely from scratch as part of a university graduation project and competed strongly in a university competition.

---

## 🚀 Features
- User-friendly interface for legal service access  
- Responsive web design for different devices  
- Organized structure for managing services  
- Simple and intuitive navigation  

---

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  
- PHP  

---

## 🎯 Purpose of the Project
The goal of this project is to demonstrate full-stack web development skills by building a real-world application that simulates a legal services platform.

---

## 🏆 Achievement
This project competed strongly in a university competition and achieved a top-ranking position.
